Now you can open `http://samlet-chat.example.com` on your browser and you should be greeted with the login page. Carry on to the [next step](/installation/self-hosting/register-your-website/README.md) to create an account and register your website.

{% hint style='tip' %}
You may want to run Commento on boot to start the server automatically with the appopriate configuration. You can do this with your favourite init system; examples files are available for [SystemD (Linux)](https://github.com/samletnorge/samlet-chat/blob/master/blob/master/etc/linux-systemd/comme2nto.service) and [rc.d (BSD)](https://github.com/samletnorge/samlet-chat/blob/master/blob/master/etc/bsd-rc/samlet-chat).
{% endhint %}
