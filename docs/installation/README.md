### Installation

As previously mentioned, there are two versions of samlet-chat &ndash; the cloud service and the self-hosted software. To install samlet-chat, choose the version that's appropriate for you and proceed from there:

 - [**Cloud Service**](cloud-service/README.md) &ndash; if you're a [samlet-chat.valiantlynx.com](samlet-chat.valiantlynx.com) customer

 - [**Self-hosting**](self-hosting/README.md) &ndash; if you're installing samlet-chat on your server
